# 🏥 Problem Statement

## 📌 Background

Healthcare organizations generate large volumes of patient, hospital, and billing data every day. However, raw healthcare data alone does not provide meaningful insights unless it is properly cleaned, analyzed, and visualized.

Hospitals often face challenges in:
- Monitoring patient admission trends
- Understanding billing patterns
- Identifying high-cost treatments and patients
- Evaluating hospital and doctor performance
- Measuring operational efficiency based on patient stay duration
- Managing healthcare costs effectively

Without proper analysis, decision-making becomes slow, inefficient, and less data-driven.

---

# 🎯 Objective

The primary objective of this project is to analyze healthcare patient records and transform raw data into actionable business insights using Python, PostgreSQL, and Power BI.

This project focuses on:
- Analyzing patient demographics and admission patterns
- Evaluating billing and insurance trends
- Measuring hospital performance
- Identifying high-cost patients and treatments
- Calculating cost efficiency based on length of stay
- Creating an interactive dashboard for data-driven decision-making

---

# 📊 Dataset Information

The dataset contains healthcare patient records including:
- Patient demographics
- Medical conditions
- Admission and discharge dates
- Hospital and doctor details
- Insurance providers
- Billing amounts
- Admission types
- Test results and medications

Total Records Analyzed:
- 55,500+ patient records

---

# 🔄 Project Workflow

1. Loaded and explored healthcare dataset using Python
2. Performed data cleaning and preprocessing
3. Uploaded dataset into PostgreSQL database
4. Renamed and standardized SQL column names
5. Converted date columns into proper SQL DATE format
6. Performed SQL analysis using aggregations, ranking, subqueries, and window functions
7. Conducted Exploratory Data Analysis (EDA)
8. Built an interactive Power BI dashboard
9. Generated business insights and performance analysis
