# 🏥 Healthcare Patient Analytics & Cost Optimization Dashboard

## 📌 Project Overview
This is an end-to-end healthcare data analytics project focused on analyzing patient data, hospital performance, and billing trends. The goal is to extract meaningful insights and support data-driven decision-making in the healthcare domain.

The project integrates Python, SQL, and Power BI to perform data analysis and build an interactive dashboard.

---

## 🎯 Objectives
- Analyze patient admission patterns and demographics  
- Evaluate hospital and doctor performance  
- Understand billing behavior and cost distribution  
- Measure cost efficiency based on length of stay  
- Visualize insights using an interactive dashboard  

---

## 🛠️ Tools & Technologies
- **Python** (Pandas, NumPy, Matplotlib, Seaborn)  
- **SQL (PostgreSQL)**  
- **Power BI**  

---

## 🐍 Python (EDA)
Performed exploratory data analysis to:
- Clean and preprocess data  
- Handle missing values and duplicates  
- Analyze distributions and relationships  
- Identify trends, patterns, and outliers  

---

## 🗄️ SQL Analysis
Used SQL to:
- Perform aggregations and grouping  
- Identify top hospitals and high-cost patients  
- Analyze revenue and billing trends  
- Apply window functions for ranking and insights  

---

## 📊 Power BI Dashboard
Developed a multi-page interactive dashboard including:

- **Executive Summary**  
- **Medical Condition Analysis**  
- **Billing & Insurance Analysis**  
- **Doctor & Hospital Performance**  
- **Cost Efficiency Analysis**  

Features:
- KPI cards (Total Patients, Billing, Avg Stay, Cost per Day)  
- Interactive filters and slicers  
- Trend and comparison analysis  

---

## 🔍 Key Insights
- Patient admissions are evenly distributed across types  
- Billing amounts vary significantly with some high-cost cases  
- Weak correlation between age, billing, and length of stay  
- A few hospitals contribute major patient volume  
- Cost efficiency varies across medical conditions  

---

## 📈 Conclusion
The project demonstrates how healthcare data can be transformed into actionable insights using data analytics tools. It highlights the importance of data-driven decision-making in improving operational efficiency and cost management.
